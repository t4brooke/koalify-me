Introduction

Game Development is all about creating interactive, engaging video games. This major combines creative storytelling, artistic design, and technical programming to bring imaginative worlds to life. You’ll learn about game mechanics, graphics, and user engagement, making it a perfect choice for anyone passionate about gaming and digital entertainment.

Careers

Game Designer – Game designers create the concepts, rules, and mechanics that make video games fun and engaging. They collaborate with artists, writers, and programmers to bring ideas to life.

Game Programmer – Game programmers write the code that powers video games, ensuring smooth gameplay, physics, and artificial intelligence. They work with game engines like Unity and Unreal Engine.

3D Artist/Animator – These artists design and animate characters, environments, and objects in video games. They use software like Blender or Maya to create visually stunning graphics.
